# VBA_StockTicker_Total_Volume_Script

This VBA script will loop through stock data listed in the excel sheet, and return the total volume traded for each stock ticker symbol over that year.

The Images folder shows screenshots of the macro output, the stock total volumes traded by ticker symbol.
The excel sheet includes the test data and the macro code.
The macro code by itself is listed as “VBA_Ticker_Volume_Script”.
